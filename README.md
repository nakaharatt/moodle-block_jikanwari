# jikanwari

日本語
===

概要
- jikanwari は Moodle のブロックプラグインです。時間割（ジカンワリ）を表示・管理するための簡易ブロックとして機能します。

主な機能
- コースページやダッシュボードに時間割を表示できます。
- 管理画面から設定を変更できます（将来的に拡張可能）。

インストール（Moodle のブロックとして）
1. Moodle のルートディレクトリにある `blocks/` ディレクトリへ本リポジトリのフォルダ全体を配置します。
   - 例: `moodle/blocks/jikanwari` のように配置します。
2. Moodle サイト管理にログインし、`サイト管理 > 通知` (Site administration > Notifications) にアクセスします。
   - プラグインのインストール・データベース更新処理が自動的に実行されます。



English
===

Overview
- jikanwari is a Moodle block plugin that provides a simple timetable display and basic management features.

Key features
- Display a timetable on course pages or the dashboard.
- Basic settings manageable from the site administration area (extendable).

Installation (as a Moodle block)
1. Place the plugin folder into Moodle's `blocks/` directory.
   - Example: `moodle/blocks/jikanwari`.
2. Log in to Moodle as an administrator and go to `Site administration > Notifications`.
   - Moodle will detect the new plugin and run the installation/upgrade steps.
